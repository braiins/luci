<%:bOSminer preview%>

<%:This is a combo preview of the new bitcoin mining software (bOSminer) and a demo of the new binary mining protocol (Stratum V2).%>

<%:The prototype has limited capabilities:%>

  • <%:it only uses 1 hashboard of the S9%>
  • <%:it doesn't have an API%>
  • <%:AsicBoost is enabled by default, it can be turned off by%> --disable-asic-boost

<%:How to run it:%>

ssh root@<%=hostname%>
root@<%=hostname%>:~# bosminer [FLAGS] --pool <HOSTNAME:PORT> --user <USERNAME.WORKERNAME>

<%:Full options:%>

bosminer -h

<%:You can connect the miner to our Stratum V2 endpoint:%>

bosminer --pool v2.stratum.slushpool.com:3336 --user <USERNAME.WORKERNAME>

<%:If you want to connect the miner to your own pool, you have to run a V2%>→<%:V1 translating proxy locally. To compile and install the proxy, please follow these instructions:%>
